## ✨ ANIMAÇÕES

### **Ao Carregar**

```css
/* Fade in + Slide up */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(40px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Aplicado na seção */
animation: fadeInUp 1s ease-out;
```

### **Badge Pulsante**

```css
/* Ponto vermelho pulsante */
@keyframes ping {
  75%, 100% {
    transform: scale(2);
    opacity: 0;
  }
}

/* Círculo interno */
animation: ping 1s cubic-bezier(0, 0, 0.2, 1) infinite;
```

### **Hover nos Passos**

```css
/* Ícone cresce ao passar o mouse */

  transition: transform 0.3s ease;
}
```

### **QR Code Card**

```css

}
```
